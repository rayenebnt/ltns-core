06 — Typographies

Trois polices, trois rôles.

Display

Space Grotesk

Pour le wordmark, les gros titres, les chiffres XL. Bold (700) avec letter-spacing -0.05em.

Mono / système

JetBrains Mono

HUD, étiquettes, readouts, métadonnées. Tabular-nums activé pour aligner les chiffres.

Corps de texte

Inter Tight

Paragraphes longs, descriptions, conditions, FAQ. Lisible à toutes les tailles d'écran.

07 — Règles

À faire · à éviter.

Respecter le clear-space

Toujours laisser une marge équivalente à la hauteur du mark autour du logo. Pas d'éléments collés.

Ne pas déformer

Pas de stretch horizontal/vertical, pas de rotation, pas d'outline ajouté.

Sur photo

Utiliser uniquement la version monochrome (cream ou noir) avec un contraste suffisant.

Pas d'autre couleur

L'accent violet est non négociable. Ne pas le remplacer par du rouge, bleu, ou autre.

Le mark seul

Quand l'espace est limité (favicon, watermark, mini-format), n'utiliser que le mark.

Pas de drop-shadow

Le glow violet du mark accent suffit. Pas d'ombre portée, pas de relief, pas de gradient sur le wordmark.
